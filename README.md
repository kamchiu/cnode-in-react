
## How to use

First, clone the repo.

```bash
$ git clone git@github.com:kamchiu/cnode-in-react.git
$ cd cnode-in-react
```


Then, install the dependencies.

```bash
$ npm install
```

Next, launch the react app.

```bash
$ npm start
```

Now you will see the app is running at http://127.0.0.1:8080.
